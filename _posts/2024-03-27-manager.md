<br>

### 상품 정보
- 상품 리스트 페이지<br>
  <img style="width:700px;height:400px;" src='/VirtualFit/assets/img/mgr/2_productlist.png'><br>
  버추얼 핏으로 랜더링 가능한 상품의 리스트를 보여주는 페이지이며 상품의 URL을 확인할 수 있습니다.<br>
  - 기본적으로 엠파파에서 제공하는 무료 상품이 포함되어 있습니다.<br>
    <br>

  <img style="width:700px;height:400px;" src='/VirtualFit/assets/img/mgr/2_copyproductlist.png'><br>
  복사 기능을 통해 이미 등록된 상품과 동일한 상품을 추가할 수 있습니다.<br>
  <br>

- 상품 추가 페이지<br>
  <img style="width:700px;height:400px;" src='/VirtualFit/assets/img/mgr/2_addproduct.png'><br>
  버추얼 핏으로 랜더링 할 상품을 추가하는 페이지입니다.<br>
  <br>

  등록 사항<br>
  - 상품 명 : 상품 명 입력<br>
  - 썸네일 이미지 : 썸네일 이미지 파일 업로드<br>
  - 타입 1 : 카테고리 등 추가적으로 기입하고 싶은 정보 입력<br>
  - 타입 2 : 카테고리 등 추가적으로 기입하고 싶은 정보 입력<br>
  - 공개 여부 : 상품 공개 여부 선택<br>
  - 상세 정보 : 상품 상세 정보 입력<br>
  - 패턴 : 패턴 리스트에 저장되어 있는 패턴 선택<br>
  - 원단 : 원단 리스트에 저장되어 있는 원단 선택<br>
    <br>

- 상품 수정 페이지<br>
  <img style="width:700px;height:400px;" src='/VirtualFit/assets/img/mgr/2_updatecopyproduct.png'><br>
  등록된 상품을 수정하는 페이지입니다.<br>
  - 본인이 등록한 상품만 수정, 삭제 가능합니다.<br>
  - 다른 사용자가 등록한 상품은 복사본을 생성하여 수정 가능합니다.<br>
    <br>

- 원단 추가 페이지<br>
  <img style="width:700px;height:400px;" src='/VirtualFit/assets/img/mgr/4_addfabric.png'><br>
  원단을 추가하는 페이지입니다.<br>
  <br>

  등록 사항<br>
  - 원단 명* : 원단 명 입력<br>
  - 썸네일 이미지* : 썸네일 이미지 파일 업로드<br>
  - DIFFUSE MAP* : 디퓨즈맵 파일 업로드<br>
  - NORMAL MAP* : 노말맵 파일 업로드<br>
  - OCCLUSION MAP : 오클루션맵 파일 업로드<br>
  - SPECULAR MAP : 스페큘러맵 파일 업로드<br>
  - 타입 1 : 카테고리 등 추가적으로 기입하고 싶은 정보 입력<br>
  - 타입 2 : 카테고리 등 추가적으로 기입하고 싶은 정보 입력<br>
  - 공개 여부 : 원단 공개 여부 선택<br>
  - 상세 정보 : 원단 상세 정보 입력<br>
  - 혼용율 : 원단 혼용율 값 입력<br>
  - 타일링 : 원단 타일링 값 입력<br>
  - 메틸네스 : 원단 메틸네스 값 입력<br>
  - 투명도 : 원단 투명도 값 입력(0~1)<br>
  - 표면 거칠기 강도 : 원단 표면 거칠기 강도 입력(0~1)<br>
    <br>
